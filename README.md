## ✨ Key Features of This Portfolio

- **Interactive 3D Elements:** Custom 3D computer model and floating skill spheres built with `@react-three/fiber` and `@react-three/drei`.
- **Modern UI/UX:** Glassmorphism components, gradient text, and parallax scroll effects powered by `framer-motion`.
- **Fully Responsive:** Carefully crafted breakpoints ensuring a seamless experience across Mobile, Tablet, and Desktop.
- **Performance Optimized:** Lazy loading for heavy 3D assets to maintain excellent Core Web Vitals.
- **Working Contact Form:** Integrated with `EmailJS` for direct messaging.

## 📂 Project Structure

```text
src/
├── assets/        # Images, 3D models (GLTF), and SVGs
├── components/    # Reusable UI components
│   ├── canvas/    # Three.js 3D models (Computers, Earth, Balls, Stars)
│   ├── layout/    # Navbar and structural elements
│   └── sections/  # Page sections (Hero, About, Works, Contact, etc.)
├── constants/     # Static data (Projects, Skills, Experience text)
├── hoc/           # Higher Order Components for section wrappers
└── utils/         # Helper functions and motion variants
```

## 💻 Installation & Local Setup

Want to run this portfolio locally? Follow these steps:

```bash
# 1. Clone the repository
git clone https://github.com/abdelrahman-samy-dev/abdelrahman-samy-dev.git

# 2. Navigate to the project directory
cd abdelrahman-samy-dev

# 3. Install dependencies
npm install

# 4. Start the development server
npm run dev
```

Visit `http://localhost:5173` (or the port provided by Vite) in your browser.
